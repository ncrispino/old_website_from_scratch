Party Leadership in the House¶

My goal with this post is to analyze the relationship between party voting statistics and congressional leadership in the House.

Much of the analysis is elementary, but the project gives me practice with cleaning and manipulating data, as well as with communicating my code and results, which will be of great help to any future academic or data science oriented project.

def paginated_request(url, num_results):
    """
    Given a url ending with 'offset=', creates a dataframe with the specified number of results (in multiples of 20)
    """    
    dfs = []
    for i in range(0, num_results, 20):
        data = propublica_data(url + str(i))
        df = pd.DataFrame(data["results"][0])
        dfs.append(df)
    return pd.concat(dfs)

def get_congress_members(congress_sessions, chambers=["house", "senate"]):
    congress_members_dfs = []
    for con in congress_sessions:
        for ch in chambers:
            url = f"https://api.propublica.org/congress/v1/{con}/{ch}/members.json"
            df = pd.DataFrame(propublica_data(url)["results"][0]["members"])
            # add column with congressional session
            df["congress"] = con
            congress_members_dfs.append(df)
    congress_members = pd.concat(congress_members_dfs)
    # add new column to congress_members with name of form (last_name, first_name)
    congress_members["full_name"] = congress_members["last_name"] + ", " + congress_members["first_name"]
    return congress_members

def house_table(url):
    """
    Given link specifying a leadership position from US House's website (found on https://history.house.gov/People/),
    scrapes table and puts into a pandas df.
    """
    site = requests.get(url)
    soup = BeautifulSoup(site.text, "lxml")
    table = soup.find("table", {"id": "tblSort"})
    headers = []
    rows = []
    for th in table.find_all("th"):
        headers.append(th.text)
    for tr in table.tbody.find_all("tr"):
        curr_row = []
        for td in tr.find_all("td"):
            curr_row.append(td.text.strip())
        rows.append(curr_row)
    return pd.DataFrame(rows, columns=headers)

def standardize_name(x):
    """
    Removes numbers at the end of names (corresponding to superscripts in House website data) 
    and removes unnecessary capitalization.
    """
    return re.sub(r"[0-9]+$", "", x).strip().title()
# house_leaders_df["Name"] = house_leaders_df["Name"].apply(standardize_name)

def last_first_names(x):
    """
    Removes middle names and replaces firstname with nickname if applicable (as Propublica API uses nicknames)
    """
#     print(x)
    lastname = re.findall(r"^[a-zA-Z'-]+", x)[0].title()
    firstname = re.findall(r",\s([a-zA-Z'-]+)", x)[0]
    nickname = re.findall(r"\([a-zA-Z\s\.]+\)$", x)
    if len(nickname) == 0:
        return lastname + ", " + firstname
    # get nickname and remove parens and spaces
    nickname_use = re.sub(r"\s", "", nickname[0][1:-1])
    return lastname + ", " + nickname_use

Comparing Voting Statistics Among Leaders¶

Before starting, note that percent votes for or against one's party represents the percentage of votes a given member of congress voted for or against the majority of their party.

For example, from Propublica, the votes against party percent of Sen. Warren would be 21.2% in the 116th Congress: "Sen. Warren voted against a majority of Senate Democrats 92 times (21.2%) in the 116th Congress (2019-20)."

[2] Standard Errors assume that the covariance matrix of the errors is correctly specified.

By simply looking at the graph, we can tell that the regression isn't useful for our purposes, as observations are fairly distanced from the regression line. Additionally, the regression coefficient is so low that an increase of ~$34\%$ would be needed to raise the number of sessions in leadership by one. So, there isn't enough evidence to conclude there's a linear relationship between the average percentage votes with party and the number of sessions served in leadership.

What This Means¶

The House leaders I identified—4 for each party—are unsuprisingly good at voting with/not voting against their party. Democratic and Republican leadership have some differences, but are overall similar in their voting patterns. From this analysis, if we can conclude anything, we can conclude that most of the top leadership positions do their job as representatives of their party, based on voting statistics alone.
